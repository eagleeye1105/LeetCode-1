# 14. 最长公共前缀

[链接](https://leetcode-cn.com/problems/longest-common-prefix/description/)

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

**示例 1:**

```
输入: ["flower","flow","flight"]
输出: "fl"
```

**示例 2:**

```
输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。
```

**说明:**

所有输入只包含小写字母 `a-z` 。

**我的实现**

```c++
class Solution {
public:
    string longestCommonPrefix(vector<string>& strs) {
        string s("");
        
        if( strs.size() == 0 )
        	return s;
        
        for( int i = 0; i < strs[0].length(); i++ )
        {
            char temp = strs[0][i];
            
            int j = 1;
            for( ; j < strs.size(); j++ )
            {
                if( temp != strs[j][i] )
                    break;
            }
            
            if( j != strs.size() )
                break;
            
            s += temp;
        }
        
        return s;
        
    }
};
```
