# 344. 反转字符串

编写一个函数，该函数以字符串作为输入，并返回反转的字符串。

例如：

给定s = "hello", 返回 "olleh".

**思路分析**

双指针

**我的实现**

```c++
class Solution {
public:
    string reverseString(string s) {
        if(s.length()==0 || s=="") return s;
        int start = 0;
        int end = s.length()-1;
        while(start<end)
        {
            char temp = s[start];
            s[start] = s[end];
            s[end] = temp;
            ++start;
            --end;
        }
        return s;
    }
};
```
